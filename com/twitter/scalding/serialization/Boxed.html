Related Docs: object Boxed | package serialization

trait Boxed[+K] extends AnyRef

This interface is a way of wrapping a value in a marker class whose class identity is used to control which serialization we use. This is an internal implementation detail about how we interact with cascading and hadoop. Users should never care.
