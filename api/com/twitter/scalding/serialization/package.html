    RequiredBinaryComparators provide comparators (or Ordering in Scala) that are capable of comparing keys in their serialized form reducing the amount of time spent in serialization/deserialization. These comparators are implemented using Scala macros, and currently provide binary comparators for primitives, strings, Options, tuples, collections, case classes and Scrooge objects.
